# 🎮 Stackly ESports

Stackly ESports is a premier platform for competitive gaming, tournaments, and esports intelligence.

## 🌐 Live Demo
Check out the live website here: [Stackly ESports](https://lingalamanasa.github.io/ESPORTS/)

## 📁 GitHub Repository
[https://github.com/lingalamanasa/ESPORTS](https://github.com/lingalamanasa/ESPORTS)

## ✨ Features
- 🏠 Responsive Homepage with Hero Section & Animations
- 👤 Player and Admin Dashboards
- 🏆 Global Leaderboards & Statistics
- 📋 Tournament Registrations
- 📰 ESports News & Intelligence Hub
- 📝 Blog, About, Services & Contact Pages
- 🔐 Login & Registration Pages
- 🎨 Modern Glassmorphism UI with Dark Theme

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom Properties, Glassmorphism, Animations)
- JavaScript (Vanilla)
- Bootstrap Icons

## 📄 Pages
| Page | Description |
|------|-------------|
| `index.html` | Homepage with hero, stats, tournaments, players |
| `about.html` | About Stackly ESports |
| `services.html` | Services offered |
| `blog.html` | Blog & news articles |
| `contact.html` | Contact form |
| `login.html` | User login |
| `register.html` | User registration |
| `dashboard-admin.html` | Admin dashboard |
| `dashboard-player.html` | Player dashboard |
| `error.html` | 404 error page |

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/lingalamanasa/ESPORTS.git
   ```
2. Open `index.html` in your browser.
